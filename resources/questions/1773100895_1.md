Consider the following Python code snippet:

```python
import asyncio

class AsyncDecorator:
    def __init__(self, func):
        self.func = func

    def __call__(self, *args, **kwargs):
        return asyncio.run(self.func(*args, **kwargs))

@AsyncDecorator
async def async_task(x):
    await asyncio.sleep(1)
    return x * 2

async def main():
    result = await async_task(5)
    print(result)

# Run the main function
asyncio.run(main())
```

What will be the output of the above code? 

A) It will raise a TypeError because `async_task` is not properly decorated.
B) It will print "10" after approximately 1 second.
C) It will raise an asyncio.TimeoutError due to an infinite loop in `async_task`.
D) It will print nothing and terminate immediately.
    
    ## Answer
    **Answer: B** 

**Explanation:** The code defines a custom decorator `AsyncDecorator` that wraps any function to run it using `asyncio.run()`. The decorated function `async_task` is an asynchronous function that sleeps for 1 second and then returns double the input value. In the `main` function, `await async_task(5)` is called, which will asynchronously wait for `async_task` to complete and return its result. After approximately 1 second, it prints "10". Therefore, the correct output is "10" after a delay of about 1 second.
