Consider the following Python code snippet that attempts to implement a metaclass to ensure all methods of a class are asynchronous:

```python
import asyncio

def async_methods(cls):
    for attr_name, attr in cls.__dict__.items():
        if callable(attr):
            setattr(cls, attr_name, asyncio.coroutine(attr))
    return cls

@async_methods
class MyClass:
    def my_method(self):
        print("Hello, world!")

# Usage
loop = asyncio.get_event_loop()
task = loop.create_task(MyClass().my_method())
loop.run_until_complete(task)
```

This code defines a decorator `async_methods` that converts all callable attributes of a class into asynchronous methods. The `MyClass` is decorated with `@async_methods`, and it includes a method `my_method`.

**Question**: What is the issue with the above code, and how would you fix it to make it work correctly?

A) The decorator does not properly handle instance methods.
B) The `asyncio.coroutine` function has been deprecated since Python 3.5 and should be replaced with `async def`.
C) The usage of `loop.create_task()` and `loop.run_until_complete()` is incorrect.
D) None of the above.
    
    ## Answer
    Part 2 (Answer):
**A) The decorator does not properly handle instance methods.**

Explanation: The code attempts to convert all callable attributes into async methods, but it doesn't distinguish between class methods and instance methods. In Python, when you use `async def` inside a class, it implicitly becomes an instance method. However, the code in the decorator uses `callable(attr)` which does not differentiate between these types of methods. As a result, when you try to call the method on an instance (`MyClass().my_method()`), it will raise an error because it expects an instance method but receives a coroutine function.

To fix this issue, you need to ensure that only instance methods are converted into async methods while class methods remain unaffected. This can be achieved by checking if the attribute is bound to an instance when it is accessed:

```python
def async_methods(cls):
    for attr_name in dir(cls):
        attr = getattr(cls, attr_name)
        if callable(attr) and not isinstance(attr, classmethod) and not isinstance(attr, staticmethod):
            setattr(cls, attr_name, asyncio.coroutine(attr))
    return cls

@async_methods
class MyClass:
    def my_method(self):
        print("Hello, world!")

# Usage
loop = asyncio.get_event_loop()
task = loop.create_task(MyClass().my_method())
loop.run_until_complete(task)
```
