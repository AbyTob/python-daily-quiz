```python

            raise TypeError("Class does not implement required methods")
        return super().__new__(cls, name, bases, dct)

class InterfaceBase(metaclass=InterfaceMeta):
    pass

class ConcreteClass(InterfaceBase):
    def method1(self):
        print("Method 1 implemented")

```

Explanation: When a class inherits from `InterfaceBase`, it must implement both `method1` and `method2`. Since `ConcreteClass` implements both methods, it is correctly enforced by the metaclass at class creation time, allowing it to be instantiated without any errors.

Explanation: Incorrect. The `InterfaceMeta` metaclass enforces the presence of required methods during class creation (`__new__`) rather than at runtime.
