Consider the following Python code that uses a metaclass to automatically add a `__repr__` method to any class it decorates, which simply prints out the class name.

```python
class AutoReprMeta(type):
    def __new__(cls, name, bases, dct):
        dct.setdefault("__repr__", lambda self: f"{self.__class__.__name__}()")
        return super().__new__(cls, name, bases, dct)

class MyClass(metaclass=AutoReprMeta):
    pass

obj = MyClass()
print(obj)
```

What will be the output of `print(obj)` when this code is executed?

A) `<MyClass object at 0x7f8b9c4d5e20>`

B) `MyClass()`

C) An error because `__repr__` is not defined

D) `None`

[ANSWER]
B) `MyClass()`

Explanation: The metaclass `AutoReprMeta` is designed to automatically add a `__repr__` method to any class it decorates. This method, when called, returns a string representation of the object in the format `{ClassName}()`. When an instance of `MyClass` is created and printed, the `__repr__` method defined by the metaclass is used, resulting in the output `MyClass()`.
