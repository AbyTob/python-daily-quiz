    ## Question
    [QUESTION] Consider the following Python code snippet:

```python
import asyncio

async def task(name, delay):
    print(f"Task {name} started")
    await asyncio.sleep(delay)
    print(f"Task {name} finished")

async def main():
    tasks = [task(f'Task_{i}', i) for i in range(5)]
    await asyncio.gather(*tasks)

# Run the event loop
asyncio.run(main())
```

Which of the following statements is true regarding the execution order and behavior of this code?

A) The tasks will run concurrently, but there is no guaranteed order.

B) The tasks will run serially in the order they are created.

C) The tasks will run concurrently, but Task_0 finishes before any other task.

D) An error will be raised because the `main` function does not return an awaitable object.
    
    ## Answer
    A) The tasks will run concurrently, but there is no guaranteed order.
