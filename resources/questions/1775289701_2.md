Consider the following Python code using asyncio:

```python
import asyncio

class AsyncWorker:
    def __init__(self, name):
        self.name = name
    
    async def perform_task(self):
        print(f"{self.name} started")
        await asyncio.sleep(1)
        print(f"{self.name} finished")

async def main():
    workers = [AsyncWorker(f"Worker{i}") for i in range(3)]
    tasks = [worker.perform_task() for worker in workers]
    await asyncio.gather(*tasks)

if __name__ == "__main__":
    asyncio.run(main())
```

Which of the following statements is true about this code?  
A) The workers will run concurrently.  
B) Each worker will wait for the previous one to finish before starting.  
C) The program will hang indefinitely because of an infinite loop.  
D) All tasks will be executed in a single thread.
    
    ## Answer
    [ANSWER] A) The workers will run concurrently.

Explanation: In the given code, `asyncio.gather(*tasks)` is used to run all tasks concurrently. Each worker's `perform_task` method is an asynchronous function that simulates a task taking 1 second to complete using `await asyncio.sleep(1)`. Since `asyncio.gather` runs these tasks concurrently, all workers will start and finish their tasks at the same time or in parallel, not waiting for each other.
